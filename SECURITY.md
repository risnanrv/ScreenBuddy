# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

ScreenBuddy takes software security seriously. Because ScreenBuddy runs entirely offline without any network capabilities, security risks are significantly lower by design.

If you discover a security vulnerability or potential crash/privilege defect, please report it via private GitHub Security Advisory or directly to the repository maintainers.

Please do **not** report security vulnerabilities via public GitHub issues.
